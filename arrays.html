در Go، یک آرایه دنباله‌ای شماره‌گذاری شده از عناصر با طول مشخصه. در کدهای معمولی Go، اسلایس‌ها خیلی رایج‌تر هستن؛ آرایه‌ها در بعضی از سناریوهای خاص مفید هستن.

اینجا ما یک آرایه a می‌سازیم که دقیقاً 5 int رو نگه می‌داره. نوع عناصر و طول هر دو بخشی از نوع آرایه هستن. به طور پیش‌فرض، مقدار آرایه صفره، که برای intها یعنی 0.

 
    var a [5]int
    fmt.Println("emp:", a)

می‌تونیم با استفاده از دستور array[index] = value یک مقدار رو در یک ایندکس تنظیم کنیم و با array[index] مقدار رو بگیریم.

 
    a[4] = 100
    fmt.Println("set:", a)
    fmt.Println("get:", a[4])

تابع داخلی len طول آرایه رو برمی‌گردونه.

 
    fmt.Println("len:", len(a))

از این دستور برای تعریف و مقداردهی آرایه در یک خط استفاده کنید.

 
    b := [5]int{1, 2, 3, 4, 5}
    fmt.Println("dcl:", b)

همچنین می‌تونید به کامپایلر اجازه بدین که تعداد عناصر رو برای شما بشماره با ...

 
    b = [...]int{1, 2, 3, 4, 5}
    fmt.Println("dcl:", b)

اگه ایندکس رو با : مشخص کنید، عناصر بین اون‌ها صفر می‌شن.

 
    b = [...]int{100, 3: 400, 500}
    fmt.Println("idx:", b)

تایپ آرایه‌ها یک‌بعدی‌اند ولی می‌تونید تایپ‌های مختلفی رو ترکیب کنید تا ساختارهای داده چندبعدی بسازید.

 
    var twoD [2][3]int
    for i := 0; i < 2; i++ {
        for j := 0; j < 3; j++ {
            twoD[i][j] = i + j
        }
    }
    fmt.Println("2d: ", twoD)

همچنین می‌تونید آرایه‌های چندبعدی رو در یک مرحله تعریف و مقداردهی کنید.

 
    twoD = [2][3]int{
        {1, 2, 3},
        {1, 2, 3},
    }
    fmt.Println("2d: ", twoD)
}

توجه داشته باشید که وقتی آرایه‌ها با fmt.Println چاپ می‌شن، به صورت [v1 v2 v3 ...] نمایش داده می‌شن.
