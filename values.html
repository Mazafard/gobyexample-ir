گو با مثال: Values

Go انواع مختلفی از مقادیر مثل رشته‌ها، اعداد صحیح، اعشاری، بولین‌ها و غیره داره. اینجا چند مثال ساده داریم.

 
package main
 
import "fmt"

رشته‌ها (Strings)، که می‌تونن با + به هم اضافه بشن.

 
    fmt.Println("go" + "lang")

اعداد صحیح و اعشاری.

 
    fmt.Println("1+1 =", 1+1)
    fmt.Println("7.0/3.0 =", 7.0/3.0)

بولین‌ها (Booleans)، با عملگرهای بولی همونطوری که انتظار دارین.

 
    fmt.Println(true && false)
    fmt.Println(true || false)
    fmt.Println(!true)
}
