گو با مثال: Hello World

برنامه اولمون پیام معروف “hello world” رو چاپ می‌کنه. این هم کل کدش.

 
package main
 
import "fmt"
 
func main() {
    fmt.Println("hello world")
}

برای اجرای برنامه، کد رو تو فایل hello-world.go بذارین و از دستور go run استفاده کنین.

 
$ go run hello-world.go
hello world

بعضی وقت‌ها می‌خوایم برنامه‌هامون رو به صورت باینری بسازیم. این کار رو با استفاده از go build انجام بدیم.

 
$ go build hello-world.go
$ ls
hello-world    hello-world.go

بعدش می‌تونیم فایل باینری ساخته شده رو مستقیم اجرا کنیم.

 
$ ./hello-world
hello world

حالا که می‌تونیم برنامه‌های ساده Go رو اجرا و بیلد کنیم، بیاین بیشتر درباره این زبان یاد بگیریم.
