گو با مثال: Constants

Go از ثابت‌ها با مقادیر کاراکتری، رشته‌ای، بولین و عددی پشتیبانی می‌کنه.

 
package main
 
import (
    "fmt"
    "math"
)

const یک مقدار ثابت رو تعریف می‌کنه.

 
const s string = "constant"
 
func main() {
    fmt.Println(s)

دستور const هر جایی که دستور var میاد، می‌تونه بیاد.

 
    const n = 500000000

عبارات ثابت با دقت دلخواه محاسبات انجام می‌دن.

 
    const d = 3e20 / n
    fmt.Println(d)

یک ثابت عددی (numeric constant) تا زمانی که تایپی به آن اختصاص داده نشود، هیچ تایپ مشخصی ندارد. به عبارت دیگر، ثابت‌های عددی در Go به طور پیش‌فرض بدون تایپ هستند و تنها زمانی تایپ می‌گیرند که به صراحت به یک تایپ تبدیل شوند.

 
    fmt.Println(int64(d))

تایپ یک عدد بر اساس کانتکستی که در حال استفاده است می‌تونه شکل بگیره، مثلن توی تخصیص به متغیر یا فراخوانی تابع. برای مثال، اینجا math.Sin انتظار float64 داره.

 
    fmt.Println(math.Sin(n))
}
