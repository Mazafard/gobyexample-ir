گو با مثال: Variables

در Go، متغیرها به صورت صریح تعریف می‌شن و توسط کامپایلر استفاده می‌شن تا مثلاً نوع درست فراخوانی توابع رو بررسی کنه.

 
package main

var یک یا چند متغیر رو تعریف می‌کنه.

 
    var a = "initial"
    fmt.Println(a)

می‌تونین چند متغیر رو با هم تعریف کنین.

 
    var b, c int = 1, 2
    fmt.Println(b, c)

Go نوع متغیرهای مقداردهی شده رو حدس می‌زنه.

 
    var d = true
    fmt.Println(d)

متغیرهایی که بدون مقداردهی اولیه تعریف می‌شن مقدار صفر می‌گیرن. مثلاً مقدار صفر برای int برابر 0 است.

 
    var e int
    fmt.Println(e)

دستور := یک روش کوتاه برای تعریف و مقداردهی متغیر است، مثلاً توی var f string = "apple" . این دستور فقط داخل توابع قابل استفاده است.

 
    f := "apple"
    fmt.Println(f)
}
